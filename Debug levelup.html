FAQ

Common questions

Is DebugQuest free to start?+
Yes! Free tier includes 5 game challenges per day and all 90+ MCQ questions. No credit card required. The AI Robot Analyzer requires Pro since it uses Claude's API.
How does the GitHub/Google login work?+
Clicking GitHub or Google redirects you to the official OAuth login page for that provider. After you authenticate, you're redirected back to DebugQuest. In this demo, the redirect is simulated — a real deployment needs Firebase Auth or similar backend.
How does the AI Robot Analyzer work?+
The robot sends your code to Claude (Anthropic's AI) which analyzes it for syntax errors, logical bugs, and performance issues. It returns structured feedback with explanations and fix suggestions in under 2 seconds.
What languages are supported?+
Python, JavaScript, Java, C++, C, HTML/CSS/Web, SQL, and Rust. Each language has dedicated MCQ questions and debugging challenges. More languages added regularly.
Is this good for technical interview prep?+
Absolutely. Debugging and DSA are tested in FAANG and top tech interviews. DebugQuest's challenges mirror real code review bugs. The DSA and OOP MCQs cover common interview theory topics.
Can I use it on mobile?+
Yes, the website is fully responsive. An Android app via Capacitor is in development. iOS support is planned. The MCQ quiz works especially well on mobile.
